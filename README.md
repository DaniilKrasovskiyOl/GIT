Чтобы создать репозиторий, нужно прописать команду:
> **git init** - создание пустого репозитория

Чтобы показать текущее состояние репозитория, используется:
> **git status**  

Он показывает:
1. Какие файлы были изменены,
2. Что добавлено в коммит,
3. Что предстоит в коммит

**Перенос строчки в Гите - два пробела в текущей строчке и *Enter*. Увы :(**

Чтобы сохранить фаайл и добавить изменения в отслеживание, используем:

> 1. **git add filename** - добавить конкретный файл  с именем **filename** в отслеживание
> 2. **git add .** - добавить все изменения во всех файлах (все файлы в репозитории, которые менялись)
