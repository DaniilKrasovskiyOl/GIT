Чтобы создать репозиторий, нужно прописать команду:
> **git init** - создание пустого репозитория

Чтобы показать текущее состояние репозитория, используется:
> **git status**  

Он показывает:
1. Какие файлы были изменены,
2. Что добавлено в коммит,
3. Что предстоит в коммит

**Перенос строчки в Гите - два пробела в текущей строчке и *Enter*.**

Чтобы сохранить файл и добавить изменения в отслеживание, используем:

> 1. **git add filename** - добавить конкретный файл  с именем **filename** в отслеживание
> 2. **git add .** - добавить все изменения во всех файлах (все файлы в репозитории, которые менялись)

Чтобы зарегистрировать добавление вышеприведенных файлов необходимо выполнить следующую команду:

> **–a** означает: добавить все изменения в индекс до передачи.
> **-m**: сообщение.

Иногда возникает ситуация, когда вы работаете на одной ветке и вам срочно нужно переключиться на другую ветку, при этом регистрировать изменения нежелательно. Для таких случае существует команда stash. Команда stash позволит сохранить изменения, не регистрируя их. 

> **git stash**

Чтобы создать новую ветку выполните команду:

> **git checkout –b hotfix**
